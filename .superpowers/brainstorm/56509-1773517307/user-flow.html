User Flow & Pages

Every page in the app. v1 had 24 pages — here's v2.

Complete Page Map
PLAYER PAGES (4)
─────────────────────────────────────────────
/                   → Join / Sign In
                      Enter name + PIN to join (new player)
                      Enter name + PIN to resume (returning player)

/picks              → Pick Wizard
                      Category-by-category predictions
                      Same sidebar nav as v1 (collapsible on mobile)
                      Auto-saves on selection
                      Locked banner if picks are locked

/leaderboard        → Live Leaderboard
                      Real-time scores via WebSocket
                      Emoji reactions (🔥😍😱💀)
                      Connection status indicator

/my-picks           → Review Picks (read-only)
                      See your picks vs actual winners
                      Available during live + after completion


ADMIN PAGES (2)
─────────────────────────────────────────────
/admin              → Import & Setup
                      Wikipedia URL input → preview → import
                      View imported categories/nominations
                      Lock/unlock picks button
                      Reset game button

/admin/live         → Mark Winners
                      Mobile-friendly category cards
                      Tap nomination to mark as winner
                      Pick counts shown per nomination
                      Clear winner button for corrections

What Changed From v1

v1 (24 pages) v2 (6 pages)
Sign-in, Sign-up, Signup Callback (3 pages) / — just name + PIN (1 page)
Dashboard, Join Game flow (2 pages) Gone — single game, no dashboard needed
Pick Wizard (1 page) /picks — same concept, simplified (1 page)
Leaderboard (1 page) /leaderboard — same concept (1 page)
18 admin pages (events, games, works, people, categories, nominations, import) 2 admin pages (import+setup, live ceremony)

Key Details

  • Admin access: simple — could be env var with admin PIN, or a hardcoded admin name
  • /my-picks: lets players review their picks during/after the ceremony without cluttering the leaderboard
  • Auto-redirect: if picks are locked and player hits /picks, redirect to /leaderboard
  • No signup callback, no join flow — name + PIN and you're in
